# Heart-Attack-Risk-Prediction-
This project implements a Heart Disease Risk Prediction system using Machine Learning algorithms to classify patients into High Risk or Low Risk categories based on health parameters. The model uses K-Nearest Neighbors (KNN) and Naive Bayes classifiers and compares their performance using standard evaluation metrics.

#Objectives

-Apply machine learning classification techniques
